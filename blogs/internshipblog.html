Internship Experience at a Startup: Building AI-Driven Retail Systems

Software Engineer Intern · Craft AI Technologies
July 2025 – December 2025 · Pune, Maharashtra

Introduction

Interning at a startup offers a unique learning curve—ownership, speed, and exposure to real business problems. During my tenure as a Software Engineer Intern at Craft AI Technologies, I worked on building production-grade systems that combined full-stack development, AI integration, analytics, and automation.

About the Startup Environment

Craft AI Technologies is an AI-focused startup where engineers work closely with business stakeholders and real clients. The environment demanded rapid iterations, strong ownership, and delivering systems that work reliably in production.

Tech Stack Used

  • Frontend: React (JSX), Vite, Material UI (MUI), TanStack React Table
  • Backend: Python, Flask
  • Database: SQL Server, SQLAlchemy ORM
  • AI: OpenAI APIs (Natural Language to SQL – NLSQL)
  • Infrastructure: NGINX (Reverse Proxy, Static Server, Router)
  • Testing: Postman

Project 1: Retail Administration & AI Analytics Platform

Problem Statement

Retail stakeholders lacked real-time visibility into sales, inventory, and operational data and depended heavily on technical teams for reporting.

Solution

I helped build a Retail Administration Dashboard for Manik Mobile Shopee that provided analytics, reporting, and AI-powered conversational data access.

My Contributions

  • Developed a React-based dashboard using JSX, MUI, and TanStack React Table
  • Built and tested Flask REST APIs using Postman
  • Integrated an AI chatbot using Natural Language to SQL (NLSQL) with OpenAI APIs
  • Created database views and stored procedures for optimized reporting
  • Implemented analytics pipelines using SQLAlchemy ORM

Impact

  • Enabled non-technical users to query data conversationally
  • Improved business intelligence and operational efficiency

Project 2: Retailware to Online Payments Automation & Settlement System

Problem Statement

Manual reconciliation between Retailware transactions and online payment records was time-consuming and error-prone.

Solution

Built an end-to-end automated reconciliation and settlement system.

My Contributions

  • Developed APIs for transaction matching and reconciliation
  • Automated mismatch detection and settlement suggestions
  • Designed analytics-driven workflows ensuring data traceability
  • Reduced manual reconciliation effort significantly

Infrastructure & Deployment Exposure

  • Configured NGINX as reverse proxy, static server, and router
  • Understood real-world frontend–backend–database interactions

Key Highlights

  • Startup exposure with real clients and live data
  • Built production-ready full-stack systems
  • Integrated AI into real business workflows
  • Designed analytics dashboards used by stakeholders

My Learnings

  • Writing clean, scalable, maintainable code
  • Translating business problems into technical solutions
  • Practical AI integration beyond demos
  • Database optimization and system design
  • Ownership and adaptability in startup environments

Conclusion

This internship strengthened my foundation in full-stack development, AI systems, analytics, and system design. It prepared me to contribute effectively in fast-paced, real-world engineering environments.
